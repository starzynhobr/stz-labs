Python 3.11 CustomTkinter Gemini API

Game XML Translator

A ferramenta definitiva para modders e times de localização. Automatize a tradução de arquivos de diálogo complexos mantendo a integridade estrutural do jogo.

Acessar GitHub

Funcionalidades Principais

Tradução via IA

Integração nativa com a API Google Gemini para traduções contextuais de alta qualidade, superando tradutores automáticos convencionais.

Integridade XML

O algoritmo de extração e injeção preserva meticulosamente a ordem dos nós, atributos e comentários. Seu jogo não vai quebrar.

Glossário Inteligente

Mantenha a consistência de termos (nomes próprios, locais, itens) através de múltiplos arquivos com o gerenciador de glossário integrado.

Interface Moderna

Desenvolvido com CustomTkinter para uma experiência visual agradável (Dark Mode) e fluxo de trabalho intuitivo.

Progresso em Tempo Real

Acompanhe visualmente o status da tradução de cada arquivo e do projeto como um todo.

Exportação Validada

Exportação em um clique com validação estrutural básica para garantir que o arquivo final esteja pronto para o jogo.
