STZ CSV Converter

Limpeza, normalização e conversão inteligente de listas de contatos. Prepare seus dados para o Google Contacts com precisão cirúrgica.

Acessar GitHub

Funcionalidades Principais

Limpeza Profunda

Remove caracteres inválidos, formata números de telefone (DDI) e padroniza nomes automaticamente.

Deduplicação Inteligente

Detecta e funde contatos duplicados baseando-se em números de telefone, evitando bagunça na sua agenda.

Mapeamento Flexível

Interface visual para mapear colunas do seu CSV original para os campos esperados (Nome, Telefone, Tags).

Relatórios Detalhados

Gera um relatório JSON completo com estatísticas, avisos e lista de contatos suspeitos após cada processamento.

Interface PySide6 + QML

Experiência de usuário fluida e responsiva, com processamento em background para não travar a interface.

Google Ready

Focado em gerar arquivos perfeitamente formatados para importação direta no Google Contacts.
