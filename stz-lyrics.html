STZ Lyrics

Transforme sua barra de tarefas com letras de músicas minimalistas e perfeitamente sincronizadas. Simples, leve e totalmente personalizável.

Acessar GitHub

Funcionalidades Principais

Sync Perfeito

Utiliza a enorme base de dados do LRCLIB.net para buscar letras sincronizadas linha a linha para suas músicas favoritas.

Design Minimalista

Uma única linha elegante que flutua sobre sua barra de tarefas, mostrando apenas o que importa no momento exato.

Customização Total

Painel de configurações intuitivo para ajustar fontes, tamanhos, cores, sombras e dimensões da janela.

Ajuste de Delay

O áudio está adiantado? Use o controle de offset integrado para ajustar a sincronia da letra em tempo real.

WebNowPlaying

Suporte robusto para players modernos (Spotify, YouTube, SoundCloud) via extensão de navegador.

Always on Top

Pode ser configurado para permanecer sempre visível, garantindo que você nunca perca um verso.
