Política de Privacidade

Transparência total sobre como nossos softwares funcionam.

Processamento Local

Todos os softwares desenvolvidos pelo STZ LABS (CSV Converter, XML Translator, Lyrics, Suite) são projetados para processar dados localmente na sua máquina. Não enviamos seus arquivos, contatos ou xmls para servidores proprietários.

APIs de Terceiros

Algumas ferramentas utilizam serviços externos para funcionar:

  • Google Gemini API: No Game XML Translator, se você optar por usar a tradução automática, o texto é enviado diretamente do seu computador para os servidores do Google. Sua API Key é armazenada apenas localmente.
  • LRCLIB: O STZ Lyrics consulta a base de dados pública do LRCLIB para exibir as letras.
